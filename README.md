# مصحف معلّم الترتيل المفسّر — نسخة ويب تفاعلية

تطبيق ويب بسيط (HTML/CSS/JS فقط، بدون أي اعتمادات خارجية أو سيرفر) لتصفّح المصحف
بالصورة الأصلية، مع وصول سريع لأي سورة أو جزء أو رقم صفحة.

## المحتوى
- `index.html` — الصفحة الرئيسية وكل منطق التطبيق
- `data.js` — بيانات السور (114) وبدايات الأجزاء (30) المطابقة لترقيم المصحف المطبوع
- 604 صورة لصفحات المصحف (001.jpg → 604.jpg) + cover.jpg للغلاف — كل الصور بجانب index.html مباشرة، بدون أي مجلد فرعي

ترقيم الصور يطابق **تمامًا** الرقم المطبوع أسفل كل صفحة في المصحف نفسه،
فلا حاجة لأي تحويل عند الإشارة لصفحة معيّنة.

## رفعه على GitHub Pages
1. أنشئ مستودعًا جديدًا على GitHub (Public).
2. ارفع كل محتويات هذا المجلد (index.html, data.js, وكل ملفات .jpg) لجذر المستودع.
   - ملاحظة: حجم الصور الإجمالي حوالي 148 ميجا، فالرفع عبر الواجهة قد يكون أسهل
     بضغط الملفات في ZIP ثم استخدام "Upload files"، أو عبر `git push` من جهازك:
     ```
     git init
     git add .
     git commit -m "مصحف معلّم الترتيل المفسّر - نسخة تفاعلية"
     git branch -M main
     git remote add origin <رابط-مستودعك>
     git push -u origin main
     ```
3. من إعدادات المستودع: Settings → Pages → اختر Branch: `main` والمجلد `/ (root)` → Save.
4. بعد دقيقة أو دقيقتين سيكون الموقع متاحًا على رابط مثل:
   `https://<اسم-المستخدم>.github.io/<اسم-المستودع>/`

## مميزات التنقّل
- **الانتقال برقم الصفحة**: اكتب الرقم في الخانة العليا واضغط "اذهب".
- **القائمة الجانبية** (أيقونة ☰): تبويب "السور" به بحث فوري بالاسم، وتبويب "الأجزاء" بالثلاثين جزءًا.
- **شريط التمرير السفلي**: لتصفّح سريع بالسحب عبر كل الصفحات.
- **السحب باللمس / أسهم لوحة المفاتيح** (→ سابقة، ← تالية) للتنقل صفحة بصفحة.
- **نسخ رابط الصفحة**: كل صفحة لها رابط خاص بصيغة `...#page=293` يمكن مشاركته
  أو حفظه، وعند فتحه يفتح التطبيق مباشرة على نفس الصفحة.
